# hello wrold

仓颉编程语言的后缀名称是 `·cj` ，仓颉是强类型编译语言。

仓颉使用 `main` 函数作为程序入门，`main` 函数不写函数修饰符 `func`

```cj
// helloworld.cj

main {
  println("hello world")
}
```
